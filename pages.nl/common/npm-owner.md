# npm-owner

> Manage ownership of published packages.
> Meer informatie: <https://docs.npmjs.com/cli/commands/npm-owner>.

- Add a new user as a maintainer of a package:

`npm owner add {{gebruikersnaam}} {{pakket_naam}}`

- Remove a user from a package's owner list:

`npm owner rm {{gebruikersnaam}} {{pakket_naam}}`

- List all owners of a package:

`npm owner ls {{pakket_naam}}`
