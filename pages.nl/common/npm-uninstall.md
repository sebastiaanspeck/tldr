# npm uninstall

> Remove a package.
> Meer informatie: <https://docs.npmjs.com/cli/v8/commands/npm-uninstall>.

- Remove a package from the current project:

`npm uninstall {{pakket_naam}}`

- Remove a package globally:

`npm uninstall -g {{pakket_naam}}`

- Remove multiple packages at once:

`npm uninstall {{pakket_naam1 pakket_naam2 ...}}`
